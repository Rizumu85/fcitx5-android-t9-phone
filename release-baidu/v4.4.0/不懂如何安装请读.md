# fcitx5-android-t9-phone

面向物理九键安卓智能机 / 功能机的 Fcitx5 Android 输入法。项目主要维护九键体验，其他输入方式可能存在兼容问题。

## 下载

请先确认手机架构，再下载对应的三个文件。

64 位手机（arm64-v8a）：

- `org.fcitx.fcitx5.android-4.4.0-arm64-v8a-release.apk`
- `org.fcitx.fcitx5.android.plugin.rime-4.4.0-arm64-v8a-release.apk`
- `rime-ice-t9-phone-main.zip`

32 位手机（armeabi-v7a）：

- `org.fcitx.fcitx5.android-4.4.0-armeabi-v7a-release.apk`
- `org.fcitx.fcitx5.android.plugin.rime-4.4.0-armeabi-v7a-release.apk`
- `rime-ice-t9-phone-main.zip`

本版输入法和 Rime 插件版本均为 **4.4.0**，对应九键 Rime 配置 **rime-ice-t9-phone v3.0.0**。配置内容与上一版相同，已安装 v3.0.0 的升级用户不需要重新导入。

## 首次安装

1. 安装输入法本体和 Rime 插件两个 APK。
2. 打开输入法本体，按 App 引导启用输入法。首次启动会自动选择 Rime，无需手动添加中州韵，也无需删除 English 或拼音输入法。
3. 打开任意文本框，点输入法上方【⋯】->【词库切换】->【重新部署】，创建 Rime 数据目录。
4. 解压 `rime-ice-t9-phone-main.zip`，将其中所有文件和文件夹复制到：

   `Android/data/org.fcitx.fcitx5.android/files/data/rime`

5. 再次打开【词库切换】，选择【同步】，完成后选择【拼音】。

如果找不到【词库切换】，请确认 Rime 插件已经安装，再到设置【高级】中检查插件、附加组件和输入法状态。

## 后续更新

可在【关于】->【检查更新】统一检查和安装输入法本体、Rime 插件和九键 Rime 配置。进入设置时每天最多自动检查一次；网络不可用时不会反复弹出错误。

## 主要功能

- 拼音九键、笔画九键、注音九键，可分别设置默认简体 / 繁体输出。
- 普通英文 multi-tap 与联想英语 T9；联想英语支持下一词预测和本地词库学习。
- 设置【词库管理】可搜索、添加、编辑或删除联想英语学习词。
- 符号第一页末尾提供 `↵`；在支持的聊天输入框中发送消息，在其他输入框中换行。
- 数字模式支持常用运算符和简单计算结果。
- 非文本场景透传物理键，避免游戏或模拟器键位被输入法拦截。
- 密码模式提供全键盘、数字行、输入预览和窥探功能。
- 支持实体键 / 屏幕键按键音，以及导入百度输入法 Android `.bds` 皮肤里的按键音。

## 常用按键

### 中文

- `1`：拼音分词符号；笔画横；注音 ㄅㄆㄇㄈ。
- `2..9`：输入对应拼音字母、笔画或注音分组。
- `0`：拼音 / 笔画有候选时确认，空闲时空格；注音模式输入 ㄧㄨㄩ。
- 短按 `*`：打开中文符号，再按一次切换中英文符号。
- 短按 `#`：输入编码时提交上方的拼音、笔画或注音；空闲时回车 / 搜索。
- 长按 `#`：切换中文、英文和数字模式。
- 空闲时长按 `*`：在已启用的拼音、笔画、注音方案间切换。

### 英文

- `1`：切换 `abc / Abc / ABC`。
- `2..9`：输入字母；联想英语会显示 T9 候选。
- `*`：打开英文标点。
- `0`：空格，或确认候选并加空格。
- `#`：提交当前英文后回车 / 搜索；长按切换模式。

### 数字

- `1..9,0`：输入数字。
- 长按数字键：输入常用运算符。
- `#`：回车 / 搜索；长按切换模式。
