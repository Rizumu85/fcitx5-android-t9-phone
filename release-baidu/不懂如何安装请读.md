# fcitx5-android-t9-phone

安卓企鹅输入法魔改成支持物理九键键盘的智能安卓机 / 老人机，对其他输入方式的支持可能有 bug。

## 下载

从网盘下载以下文件（先确认手机是 32 位还是 64 位）：

- 如果是 **64 位手机（arm64-v8a）**，下载这三个文件：
  - `org.fcitx.fcitx5.android-72b5a2cb-arm64-v8a-release.apk`
  - `org.fcitx.fcitx5.android.plugin.rime-72b5a2cb-arm64-v8a-release.apk`
  - `rime-ice-t9-phone-main.zip`

- 如果是 **32 位手机（armeabi-v7a）**，下载这三个文件：
  - `org.fcitx.fcitx5.android-72b5a2cb-armeabi-v7a-release.apk`
  - `org.fcitx.fcitx5.android.plugin.rime-72b5a2cb-armeabi-v7a-release.apk`
  - `rime-ice-t9-phone-main.zip`

> 说明：`rime-ice-t9-phone-main.zip` 是雾凇拼音 / 九键配置文件，32 位和 64 位都用同一个压缩包。

## 安装方法

- 在手机上安装下载下来的 APK，输入法本体和插件。

- 输入法本体，按照 App 显示的要求，开启输入法。

- 在输入法设置里进行一下操作：

  - 【插件】查看相应的 rime 插件有无被检测到。  
  - 【附加组件】查看中州韵有无被勾选，需被勾选。  
  - 【输入法】点右下角的添加，添加中州韵。可以按右上角的编辑删掉 English 和拼音输入法。

- 进入一个能输入文本的地方，打开输入法 → 设置（三个点的那个图标）→ 【< >】（两个大于小于的图标）→ **重新部署** 【必做】

- 把 `rime-ice-t9-phone-main.zip` 解压，并把里面的 **所有文件和文件夹**

- 导进安卓手机里的文件夹：

  `Android/data/org.fcitx.fcitx5.android/files/data/rime`  
  （没有的话就是忘记提前先部署了）

- 再次进入一个能输入文本的地方，打开输入法 → 设置（三个点的那个图标）→ 【< >】（两个大于小于的图标）→ **同步**

- 等待完成后，朙月拼音会变成雾凇拼音。

- 点开后选择中文九键模式。

- 恭喜你可以用了！

## 项目进度

### 支持模式

- 中文九键模式
- 英文模式（multi-tap 输入、Shift、Caps Lock）
- 数字模式
- 中文/英文模式长按输入数字
- 原本输入法已支持繁体切换

### 添加魔改的功能

- 九键按键映射
- 屏幕键盘在使用时常驻，只保留了必要的补充功能
- 长按 `#` 切换中文/英文/数字模式
- 中文模式 -> 支持拼音预测和拼音筛选栏
- 英文模式 -> 支持大小写
- 物理返回按键可删除文本
