# fcitx5-android-t9-phone 安装步骤（手机查看版）

安卓企鹅输入法魔改成支持物理九键键盘的智能安卓机 / 老人机。

====================
1) 先下载文件
====================

先确认手机是 32 位还是 64 位。

[64 位手机（arm64-v8a）下载这 3 个文件]
1. org.fcitx.fcitx5.android-4.0.0-arm64-v8a-release.apk
2. org.fcitx.fcitx5.android.plugin.rime-4.0.0-arm64-v8a-release.apk
3. rime-ice-t9-phone-main.zip

[32 位手机（armeabi-v7a）下载这 3 个文件]
1. org.fcitx.fcitx5.android-4.0.0-armeabi-v7a-release.apk
2. org.fcitx.fcitx5.android.plugin.rime-4.0.0-armeabi-v7a-release.apk
3. rime-ice-t9-phone-main.zip

说明：
rime-ice-t9-phone-main.zip 是雾凇拼音 / 九键配置文件，
32 位和 64 位都用同一个压缩包。

====================
2) 安装步骤
====================

1. 安装两个 APK（输入法本体 + rime 插件）。
2. 打开输入法本体，按 App 提示启用输入法。
3. 进入输入法设置，确认：
   - 【插件】能检测到 rime 插件
   - 【附加组件】中州韵已勾选
   - 【输入法】添加中州韵（可删除 English 和拼音输入法）
4. 进入任意可输入文本页面：
   输入法 -> 【⋯】快捷设置 -> 词库切换 / Rime 状态项 -> 重新部署（必做）
   第一次安装时这个状态项通常显示当前方案名，比如【朙月拼音】；
   如果方案名暂时为空，界面会显示【词库切换】。
5. 解压 rime-ice-t9-phone-main.zip。
6. 把解压后的所有文件和文件夹复制到：
   Android/data/org.fcitx.fcitx5.android/files/data/rime
   （如果没有这个目录，通常是前面忘了先“重新部署”）
7. 再进入任意可输入文本页面：
   输入法 -> 【⋯】快捷设置 -> 同一个词库切换 / Rime 状态项 -> 同步
8. 等待完成后，朙月拼音会变成雾凇拼音。
9. 选择中文九键模式，完成。

====================
3) 4.0.0 新功能
====================

- 新增【密码模式】：在输入法上方【⋯】快捷设置里可以手动开启；遇到密码框时也会尽量自动进入。
- 密码模式会显示数字行、全键盘字母和常用符号，适合输入密码、验证码、邮箱和账号。
- 密码模式有输入预览。有些 App 不会把密码框顶到输入法上方，现在输入法自己会显示本次键入内容。这个功能默认开启，可以在设置里关闭。
- 密码模式有窥探模式。按住眼睛图标时会临时收起键盘主体，方便看被键盘挡住的密码框、验证码图片或页面提示；松开后立即恢复。
- 按键音重做：支持屏幕键和实体键按键音、实体键单独开关、多种按键音风格，并在设置里提供快速预览。

====================
4) 九键使用逻辑（简版）
====================

[中文模式]
- 1: 无拼音时打开符号候选；有拼音时插入拼音分词符号 '；长按可输入数字 1 或选择第 1 个候选
- 2-9: ABC / DEF / ... / WXYZ；无拼音时长按输入对应数字；有拼音时长按选择对应候选
- *: 输入 *；符号候选时切换中英文符号
- 0: 空格；无拼音时长按输入数字 0；有拼音时长按选择第 10 个候选
- #: 回车/搜索；长按切换中文/英文/数字

[英文模式]
- 1: 常用标点（长按数字 1）
- 2-9: ABC / DEF / ... / WXYZ（长按输入对应数字）
- *: Shift（长按 Caps Lock）
- 0: 空格（长按数字 0）
- #: 回车/搜索（长按切换中文/英文/数字）

[数字模式]
- 1-9: 输入数字 1-9，长按输入常用运算符
- *: 输入 *，长按显示运算符快捷提示
- 0: 输入数字 0，长按输入 .
- #: 回车/搜索（长按切换中文/英文/数字）

====================
5) 当前支持
====================

- 中文九键模式
- 英文模式（multi-tap、Shift、Caps Lock）
- 数字模式（数字输入、常用运算符、简单计算结果）
- 物理按键选区模式
- 密码模式（全键盘、数字行、输入预览、按住窥探）
- 屏幕键和实体键按键音、多种按键音风格、设置内试听
- 中文/英文模式下长按输入数字
- 原输入法的繁体切换
- 物理返回键可删除文本
