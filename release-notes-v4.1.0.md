## 更新项目

### 新功能

- 新增【智能英语】模式：可以在输入法上方【⋯】快捷设置里和普通英文九键相互切换。
- 智能英语内置更完整的英文 T9 词库，按数字序列可以预测英文单词。
- 智能英语支持从实际输入中学习新单词，适合补充人名、术语和特殊词。

### 修复

- 在 Java 模拟器、游戏键位映射等非文本输入场景中，物理按键会透传给应用，不再被输入法误当作文字输入。
- 改善手机卡顿或发热时的长按误触，减少短按被识别成长按候选或数字的情况。
- 修复部分情况下重新部署 / 重新加载后只剩默认模式、Rime 状态没有刷新的问题。

---
这边总共有 4 个安装包，分为输入法本体和输入法插件，32 位版本和 64 位版本。
先查找自己手机型号看你的手机是 32 位版本还是 64 位版本。
32 位请下载 armeabi 版本的，64 位请下载 arm64 版本的。
根据自己的手机型号，再下载输入法本体和插件的 apk 并安装。

比如说我的手机是多亲 F22 Pro，是 64 位的手机，
那么我就要下载并安装这两个：
org.fcitx.fcitx5.android-4.1.0-arm64-v8a-release.apk
org.fcitx.fcitx5.android.plugin.rime-4.1.0-arm64-v8a-release.apk

安装好后需要回到主页的简介继续看安装教程。
