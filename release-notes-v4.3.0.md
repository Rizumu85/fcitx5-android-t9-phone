## 新功能

- 全面整理设置页面，将输入方式、外观、按键工具栏和高级功能按使用场景重新分组，并统一选项名称。
- 新增可视化工具栏设置：可预览、启用并拖动调整语音输入、撤销、重做等按钮顺序。
- 新增可实时交互的主题预览，可左右查看九键输入和密码键盘，并预览输入面板、编辑按钮及剪贴板条目的圆角效果。
- 语音输入可放入工具栏，也可选择空闲时长按 `0` 快速启动。

## 体验与性能

- 加入启动与九键关键路径的性能优化和基线配置，减少首次打开、首键输入、候选刷新和拼音筛选的等待。
- 优化候选、输入面板和工具栏的局部刷新，降低卡顿设备上的闪烁与无效重绘。
- 改进自定义字体、按键音包、联想英语学习词库及主题设置界面的排版和操作体验。
- 首次安装会自动选择 Rime；不再需要手动添加中州韵或删除 English、拼音输入法。

## 修复

- 修复长按 `0` 启动语音时可能连续输入多个 `0` 的问题。
- 修复主题预览闪退、页面穿模、尺寸错位，以及返回页面后主题入口顺序变化的问题。
- 修复中文候选窗口偶发无法显示、候选焦点及宽度异常和部分初始化状态不同步的问题。
- 【词库切换】只展示面向用户的中文方案，并将方案名称简化为【拼音】【笔画】【注音】。

## 安装包

请根据手机架构同时安装输入法本体和 Rime 插件：

- 64 位：`org.fcitx.fcitx5.android-4.3.0-arm64-v8a-release.apk` 与 `org.fcitx.fcitx5.android.plugin.rime-4.3.0-arm64-v8a-release.apk`
- 32 位：`org.fcitx.fcitx5.android-4.3.0-armeabi-v7a-release.apk` 与 `org.fcitx.fcitx5.android.plugin.rime-4.3.0-armeabi-v7a-release.apk`

九键 Rime 配置仍使用 `rime-ice-t9-phone-main.zip`。升级时请同时更新输入法本体和 Rime 插件。
