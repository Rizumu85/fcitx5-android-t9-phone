## 新功能

- 新增中英文手写输入。中文内置纯离线识别，可选下载 ML Kit 增强模型；英文模型下载完成后也可完全离线使用。
- 手写面板提供独立候选条、钢笔/毛笔笔触、撤销、退格、空格、逗号、表情、数字、符号、中英文切换和发送/回车，并支持实体九键快捷操作。
- 手写英文可结合【联想英语】显示预测候选；中文可选择在提交后短暂显示带声调的全部拼音读音。
- 检查更新现在可以从最新版向前查看多个新版本的更新说明，而不只显示一条最新版本。
- 文本编辑面板改为一体式方向控制盘，统一复制、剪切、粘贴、退格和剪贴板操作，并加入轻量按压回弹反馈。

## 体验与修复

- 手写笔迹改为低延迟渲染，减少毛笔模式和长笔画不跟手、松手后才补齐笔画的问题。
- 修复手写候选无法通过确认键或 `0` 提交、提交后画布未及时清空，以及进入表情/数字/符号后无法自然返回手写的问题。
- 改进手写候选和控制区在横屏、小屏及自定义字体下的间距、对齐与尺寸。
- 修复剪贴板长按菜单与文本编辑操作样式不统一的问题。
- 修正符号数字入口的 `123` 显示，并将快捷设置入口名称调整为更准确的【按键工具】。

## 安装包

本次 Release 提供输入法本体和 Rime 插件，并分别提供 32 位与 64 位版本：

- 32 位手机下载 `armeabi-v7a` 版本。
- 64 位手机下载 `arm64-v8a` 版本。

同一架构的输入法本体和 Rime 插件两个 APK 都需要安装。例如 64 位手机需要安装：

- `org.fcitx.fcitx5.android-4.5.0-arm64-v8a-release.apk`
- `org.fcitx.fcitx5.android.plugin.rime-4.5.0-arm64-v8a-release.apk`

安装完成后，请回到项目主页阅读首次安装教程。

## Rime 版本对应关系

- Rime 插件：`4.5.0`，请与输入法本体一起更新。
- 九键 Rime 配置：[rime-ice-t9-phone v3.0.0](https://github.com/Rizumu85/rime-ice-t9-phone/releases/tag/v3.0.0)。本次配置没有更新，已经使用 v3.0.0 的用户不需要重新导入或重新部署。

首次安装仍需下载 `rime-ice-t9-phone-main.zip` 并按 README 完成配置；以后可通过【关于】->【检查更新】统一管理输入法本体、Rime 插件和九键 Rime 配置。
